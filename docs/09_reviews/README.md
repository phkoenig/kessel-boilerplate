# 09_reviews — Change-Reviews

Output des Skills `change-reviewing` (`.claude/skills/change-reviewing/`).

## Zweck

Diff-basierte Reviews von Branches, Commit-Ranges oder unstaged Changes — mit Risiken, Testluecken und Folgearbeiten.

## Naming

`YYMMDD-<branch-oder-thema>.md` (z.B. `260419-theme-snapshot.md`).
Kleine Reviews bleiben im Chat und landen **nicht** hier.

## Archiv

Alte Reviews in `archive/` verschieben sobald der Branch gemerged und obsolet ist.
